Compute mean and standard deviation maps from a set of maps.

Usage:

  • [-help]: Display usage information.
  • [-inp filename_in]: Name of input file containing list of maps to consider.
  • [-out_mean filename_mean]: Name of output file for mean map.
  • [-out_std filename_std]: Name of output file for standard deviation map.
