Read a HEALPix alm fits file and apply (optionally) a beam and (optionally) a pixel window function.

Usage:

  • [-help]: Display usage information.
  • [-inp filename_alm]: Name of HEAPix alm fits file to read.
  • [-out filename_out]: Name of output HEALPix alm file to write.
  • [-beam filename_beam (optional)]: Name of input beam fits file.
  • [-fwhm fwhm (arcmin) (optional)]: Full-width-half-maximum (in arcmin) of Gaussian beam.
  • [-nside_pixel_window (optional)]: HEALPix nside of map if to be computed.
  • [-beam_ascii beam_ascii]: Logical to specify whether beam file is an ascii file.
